# Bright Minecraft Launcher

</div>

# 了解与介绍

`Bright`是一个`方便`，`快捷`，`实用`的`Minecraft启动器`，支持下载游戏核心，宣传服务器，Java运行时安装，可以让您在使用途中放松心情。支持`高度DIY`，打造属于您的`Minecraft启动器`。
